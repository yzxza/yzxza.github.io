一些非常简略的笔记。

神经网络基础

线性函数→→→每个类别的得分
$f\left ( x,w\right)+b$ w是权重参数、b是偏置参数

权重参数表示在某个类别中，某权重的重要性
w矩阵是优化而来的，需要不断改变w参数

损失函数：

$L_{i} = {\textstyle \sum_{j≠y_{i} }^{}} max(0，S_{j}-S_{y_{i} } +1)$
S_{j}表示错误 S_{y_{i} }表示正确 1是指容忍程度
要求正确类别的得分起码比错误类别多1以上

R(w) 正则化

加入正则化项缓解过拟合

Softmax分类器

将得分值转化为损失值
Sigmoid函数将的分值转化为概率

概率越接近于1，损失越小

前向传播：

$f\left ( x,w \right ) +b$ 得出损失值

反向传播：

更新w，以优化损失函数
链式求导法则

输入层表示多个特征的输入

非线性变化加在每一部矩阵计算之后

神经元个数越多越好，但不可过拟合

惩罚力度过小会导致过拟合
$\lambda =0.001 $
$\lambda =0.01 $
$\lambda =0.1$

参数个数即矩阵的大小，越大越容易过拟合
64 $\quad$ 128 $\quad$ 256 $\quad$ 112

激活函数（非线性变换）

由于sigmoid函数梯度为0时可能发生梯度消失现象，于是多采用RELU函数

参数初始化：根据随机策略来进行初始化

drop-out：杀死神经元，防止网络搜索过于复杂

计算机视觉

检测、分类与检索、超分辨率重构、医学细胞检测、OCR识别、无人驾驶、人脸识别

卷积网络：

卷积操作：旋转180度，与另外一个矩阵做点积

滑动窗口步长： 77 → 55 特征丰富
77 → 33 特征不丰富
常用为5*5较多的，但是计算效率会低

卷积核尺寸： 选择区域大小
边缘填充（+pad1）：由于中间点提取次数较多，外面加一圈，保证边界信息充分利用
卷积核尺寸：得到多少个特征图（10,20,30）

池化层：
22422464 → 11211264

最大池化，平均池化：分成每一部分选最大/平均

两次卷积，一次池化
CONV RELU CONV RELU POOL CONV RELU CONV RELU POOL CONV RELU CONV RELU POOL FC(全连接层)
带参数计算的称为一层，CONV,FC ，此网络有7层

经典网络

Alexnet：
11*11的卷积核、步长为4，不加pad，不常见

VGG:
3*3的卷积核，16/19层，pooling后弥补损失，特征翻倍，16层的效果比30都好

Resnet：
跳过某些层，把深度学习救活，层数越高，效果合理变好。

堆叠小的卷积核所需参数更少，卷积过程越多，特征提取越细致，越不会增大权重参数个数，典型案例就是VGG

递归神经网络RNN

用于自然语言处理
word2vec
50-300维向量
相似的词在特征表达中比较相似

CBOW:上下文→中心词
Skip-gram：中心词→上下文

A→B(十万)
A,B→0.9
负采样模型，由于target全为1，需要加入target全为0的词，target全为0的词不在语料库中，负样本是人为构建的
